What do the value symbols mean?

Every item shows its actual rated value. A flat value like 20k means the item is worth exactly that — it is not a range. Some items are rated as a real range (20k-25k), and a trailing + means the value shown is a typical average the item can exceed.

Symbol reference

O/C — Owner's Choice

O/C items have no fixed value: the owner decides what they're worth in a trade. An item marked O/C always displays “O/C” regardless of any stored value.

! — Unstable value

A reddish value with a ! marker means the price is volatile: the item was recently added, recently returned to Gamenight, or is inflated, so its value hasn't settled yet. Trade these with extra care.
